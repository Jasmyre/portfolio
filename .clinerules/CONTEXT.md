# Project Context for Cline

## Overview
This portfolio website showcases the work and skills of Jasmyre Andrei Lanuza. As an assistant, you should be familiar with the project structure and conventions to effectively contribute to development.

## Key Project Characteristics
- **Type**: Personal portfolio website
- **Framework**: Next.js App Router
- **Styling**: Tailwind CSS with shadcn/ui components
- **State Management**: TanStack Query + tRPC
- **Database**: Prisma ORM (PostgreSQL)

## Important Patterns
1. **Theme Switching**: Implemented via `next-themes` and `ThemeProvider`
2. **Component Organization**: 
   - Reusable components in `src/components`
   - UI primitives in `src/components/ui`
3. **API Routes**: tRPC routers in `src/server/api/routers`
4. **Environment Configuration**: Validated with `@t3-oss/env-nextjs`

## Common Tasks
- Adding new projects to the portfolio
- Updating skills section with new technologies
- Implementing new UI components
- Optimizing performance
- Fixing accessibility issues

## Conventions to Follow
- Branch naming from `.clinerules`
- Conventional commit messages
- TypeScript strict mode
- Tailwind utility classes for styling
- Component-driven development
