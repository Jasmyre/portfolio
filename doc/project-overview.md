# Project Overview: Jasmyre Portfolio

## Purpose
This portfolio website showcases the work and skills of Jasmyre Andrei Lanuza, a full-stack web developer. The site serves as a professional showcase for potential employers and clients.

## Key Features
- Responsive design for all device sizes
- Dark/light theme switching
- Animated UI components
- Project showcase with descriptions
- Skills section with technology icons
- Contact form functionality

## Technical Goals
- Implement modern web development practices
- Demonstrate proficiency with T3 Stack and Next.js Framework
- Showcase clean, maintainable code
- Optimize for performance and SEO

## Project Status
- Live at: [jazmyre.vercel.app](https://jazmyre.vercel.app)
- Actively maintained with new projects added regularly
- Follows semantic versioning (currently v0.1.0)
